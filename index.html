This is for my notes so that I remember how to deploy site to github!

  1. Create github repository
  2. Open terminal
  3. ls & cd into project folder
  4. git add (avoid using git add .)
  5. git status to see if there's an exsiting repository (for this example there wasn't)
  6. git init to create a new git repository
  7. git status - will show branch, commits if any, and untracked files to add or tracked files
  8. git add
  9. git status again to see what you have added/removed/etc
  10. git commit -m " "
  11. git push - if error is no push destination, go to github copy git remote add origin
  12. git branch
  13. paste git remote add origin that you copied from github
  14. git push - if error is brnach main has no upstream, follow prompt, copy and paste the provided code
  15. On github: settings, pages, select deploy from branch and change none under branch section to main, then click save and wait for site.
